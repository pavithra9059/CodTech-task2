Project Overview

The Animated Calculator is a desktop GUI application designed to perform fundamental mathematical operations such as addition, subtraction, multiplication, and division. The application combines functionality with an engaging visual design, making it an excellent beginner project for learning Python GUI development.

✨ Features
Addition of two numbers
Subtraction of two numbers
Multiplication of two numbers
Division of two numbers
Animated multicolor title
Interactive button hover effects
Attractive dark-themed interface
Error handling for invalid inputs
Division-by-zero protection
Real-time result display
🛠 Technologies Used
Python 3.x
Tkinter GUI Library
Tkinter MessageBox
